# modelagem2026

Website de exposição com conteúdo editorial, viewer 3D e realidade aumentada.

## Stack

- Next.js 15 + React 19
- HeroUI v3 + Tailwind CSS v4
- `<model-viewer>` para modelos GLB
- AR.js + A-Frame para experiência com marcador Hiro

## Documentação

- [`PLAN.md`](./PLAN.md) — arquitetura e roadmap
- [`DESIGN.md`](./DESIGN.md) — design system (tema museu)

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Deploy

Projeto configurado para Vercel. HTTPS é obrigatório para câmera e AR.
